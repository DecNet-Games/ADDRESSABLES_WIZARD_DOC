# ❓ Support & FAQ

Find answers to common questions and technical issues.

---

## 🏗️ Frequently Asked Questions

### 1. Does this tool replace Unity's Addressables package?
**No**. Addressables Wizard is an **extension** for the official package. It automates your configuration and provides a simplified management UI.

### 2. Can I use this for my existing Addressables project?
**Yes**. The Setup Wizard and Dashboard are fully compatible with existing `AddressableAssetSettings`. 
- **Recommendation**: Run the "Setup Wizard" to verify your current paths and groups for optimized health.

### 3. What platforms are supported?
Addressables Wizard supports **every platform** that Unity Addressables supports:
- Android
- iOS
- PC / Mac / Linux
- PlayStation / Xbox / Switch
- WebGL

### 4. Is there a runtime performance cost?
**0%**. All logic is contained within the `Editor` folder and stripping of unnecessary code is handled by the ASMDEF.
