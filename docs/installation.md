# 📥 Installation

Addressables Wizard is a premium Unity Editor extension. It is designed to be an all-in-one solution that manages its own dependencies and configuration.

> [!IMPORTANT]
> **Paid Asset Policy**: This tool is available exclusively through the **Unity Asset Store**. It is not a free/open-source tool. Unauthorized distribution is strictly prohibited.

---

## 🛠️ Step-by-Step Setup

### 1. Purchase & Add to Assets
Visit the [Unity Asset Store](https://assetstore.unity.com/) and purchase **Addressables Wizard**. Once purchased, click "Add to My Assets."

### 2. Import via Package Manager
1. In your Unity Project, navigate to **Window > Package Manager**.
2. Select **Packages: My Assets** from the dropdown.
3. Locate **Addressables Wizard** and click **Download**, then **Import**.

### 3. Automatic Dependency Check
Upon import, the tool will automatically check for the official Unity `Addressables` package. 
- If found: You are ready to go!
- If missing: A prompt will appear asking if you'd like the Wizard to install it for you automatically.

---

## 🚀 Post-Installation
After importing, you will see a new menu item: `Tools > DecNet > Addressables Wizard`. 

Clicking this will open the **Setup Wizard**, which will walk you through the initial configuration of your local and remote assets.
