# 🚀 Core Features

Addressables Wizard is designed to replace the standard, manual Unity Addressables windows with a faster, more intelligent management suite.

### Use Cases:
1.  **Mobile Games**: Quickly switch between ODR (On-Demand Resources) and Remote CDN assets.
2.  **Large Projects**: Bulk-index thousands of assets in seconds.
3.  **Cross-Platform Builds**: Automate platform-specific compilation with clean environments.
